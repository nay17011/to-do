# Todo App

A full stack todo application built with Python, Flask, SQLite, HTML, CSS and JavaScript.

## Status
Work in progress — backend complete, frontend coming soon.

## Project Structure
```
todo/
├── backend/
│   ├── app.py        # Entry point — Flask routes and server setup
│   ├── service.py    # Business logic layer
│   ├── models.py     # Database models and schema
│   └── todo.db       # SQLite database (auto-created on first run)
└── frontend/
    ├── index.html
    ├── style.css
    └── script.js
```

## Tech Stack
- **Python** — core language
- **Flask** — web framework
- **SQLite** — lightweight database
- **HTML, CSS, JavaScript** — frontend

## How to Run (Backend)
```
cd backend
python app.py
```

Server runs at `http://localhost:5000`
